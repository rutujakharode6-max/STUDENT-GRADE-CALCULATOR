# Student Grade Calculator

A pure Python command-line application that calculates student grades, generates class statistics, and exports beautifully formatted text-based reports.

## Features
- Interactive CLI Menu System
- Input validation (0-100 range bounds, integer requirements)
- Generates dynamic, color-coded terminal tables
- Exports timestamped `.txt` class reports
- Modular Python architecture with strict `try-except` error handling
- Zero external dependencies

## How to Run
```bash
python grade_calculator.py
```
